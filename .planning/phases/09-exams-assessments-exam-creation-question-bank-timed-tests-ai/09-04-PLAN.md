<task type="auto">
  <name>Task 1: Create grading services (ExamGradingService and AiGradingService)</name>
  <files>app/Services/ExamGradingService.php, app/Services/AiGradingService.php, config/services.php, .env.example</files>
  <action>
    Create grading services for automatic and AI-assisted grading:

    1. **Service: AiGradingService.php**
       - Create in `app/Services/`
       - Purpose: Grade short answer questions using OpenAI API
       - Constructor: accepts `api_key` (from config), `model` (from config, default 'gpt-4o-mini')
       - Method: `gradeShortAnswer(string $questionText, string $correctAnswer, string $studentAnswer, ?string $explanation = null): array`
         - Calls OpenAI Chat Completions API with prompt:
           "Grade this short answer: Question: {questionText}. Correct Answer: {correctAnswer}. Student Answer: {studentAnswer}. Explanation: {explanation}. Score from 0 to 1 (1=fully correct, 0=completely wrong). Return JSON: {score: float, reasoning: string}"
         - Parses JSON response
         - Returns: `['score' => float, 'reasoning' => string, 'provider' => 'openai']`
       - Handle API errors gracefully (return score 0 with error message)
       - Check config `exam.grading.ai_enabled` to disable AI and return manual grading needed

    2. **Service: ExamGradingService.php**
       - Create in `app/Services/`
       - Constructor: inject `AiGradingService $aiGradingService`
       - Method: `gradeAttempt(ExamAttempt $attempt): array`
         - Loops through attempt->answers with loaded examQuestion and question
         - Calls appropriate grading method based on question type
         - Updates answer: `is_correct`, `marks_awarded`
         - After all answers graded: calls `calculateTotalScore($attempt)`
         - Returns: `['total_score' => int, 'passed' => bool, 'answers' => []]`
       - Method: `gradeMcq(ExamAnswer $answer): void`
         - Get question's correct options (where is_correct=true)
         - Compare selected_option_id with correct option(s)
         - Set is_correct = true if match, marks_awarded = effective_marks or 0
       - Method: `gradeTrueFalse(ExamAnswer $answer): void`
         - Same logic as MCQ (True/False are single-correct MCQ)
       - Method: `gradeShortAnswer(ExamAnswer $answer): void`
         - If AI enabled: call `$this->aiGradingService->gradeShortAnswer()`
         - Calculate marks: effective_marks * score (0 to 1)
         - Set is_correct = score > 0.5, marks_awarded = rounded(marks * score)
         - If AI disabled: set is_correct = null (pending manual grading)
       - Method: `gradeCodeSnippet(ExamAnswer $answer): void`
         - For now: set is_correct = null (manual grading required)
         - Future: could integrate code execution sandbox
       - Method: `calculateTotalScore(ExamAttempt $attempt): void`
         - Sum marks_awarded from all answers
         - Get exam passing_score
         - Set attempt: total_score, passed (total_score >= passing_score * total_questions / 100)
         - Save attempt

    3. **Config: config/services.php**
       - Add OpenAI configuration:
         ```php
         'openai' => [
             'api_key' => env('OPENAI_API_KEY'),
             'model' => env('OPENAI_MODEL', 'gpt-4o-mini'),
         ],
         ```

    4. **.env.example**
       - Add:
         ```
         OPENAI_API_KEY=
         OPENAI_MODEL=gpt-4o-mini
         ```

    5. **Config: Create config/exam.php** (new file)
       - Settings:
         ```php
         return [
             'grading' => [
                 'ai_enabled' => env('EXAM_AI_GRADING_ENABLED', false),
                 'passing_score_default' => 50,
             ],
         ];
         ```

    Run `vendor/bin/pint --dirty` after creating files.
