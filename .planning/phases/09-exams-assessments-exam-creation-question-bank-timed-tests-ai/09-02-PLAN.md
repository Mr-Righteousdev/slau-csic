    Add methods to ExamService for managing exam questions:

    1. **addQuestion(Exam $exam, int $questionBankQuestionId, ?int $customMarks = null): ExamQuestion**
       - Create ExamQuestion record linking exam to QuestionBankQuestion
       - Set custom_marks if provided, otherwise null (use question default)
       - Set order to current max order + 1
       - Return created ExamQuestion

    2. **removeQuestion(Exam $exam, int $examQuestionId): bool**
       - Find ExamQuestion where exam_id = exam->id and id = examQuestionId
       - Delete the record
       - Reorder remaining questions (update order sequentially)
       - Return true on success

    3. **reorderQuestions(Exam $exam, array $orderedIds): void**
       - Loop through $orderedIds with index
       - Update each ExamQuestion order to index value
       - Maintain ExamQuestion belongsTo relationship

    4. **getAvailableQuestions(Exam $exam, ?string $search = null)**
       - Query QuestionBankQuestion that are NOT already in this exam
       - Filter by search on question_text if provided
       - Filter by type if provided
       - Return with options for display
       - Follow pattern from QuestionBank Index query

    5. **getExamTotalMarks(Exam $exam): int**
       - Sum effective_marks across all ExamQuestion records
       - Use ExamQuestion::getEffectiveMarksAttribute()
