    Create database schema and model for certificate eligibility:

    1. **Migration: Create certificate_eligibilities table**
       - Use `php artisan make:migration create_certificate_eligibilities_table`
       - Fields: `id`, `exam_attempt_id` (foreignId, cascade delete, unique), `user_id` (foreignId, cascade delete), `exam_id` (foreignId, cascade delete), `eligible` (boolean, default true), `notes` (text, nullable - admin notes), `created_at`, `updated_at`
       - Unique on `exam_attempt_id` (one eligibility per attempt)
       - Index on `user_id`, `exam_id`, `eligible`

    2. **Model: CertificateEligibility.php**
       - Use `php artisan make:model CertificateEligibility`
       - Fillable: `exam_attempt_id`, `user_id`, `exam_id`, `eligible`, `notes`
       - Casts method: `eligible` (boolean)
       - Relationships:
         - `examAttempt(): BelongsTo` to ExamAttempt
         - `user(): BelongsTo` to User
         - `exam(): BelongsTo` to Exam
       - Scopes: `scopeEligible($query)` where eligible = true

    3. **Service: CertificateService.php**
       - Create in `app/Services/`
       - Methods:
         - `createEligibility(ExamAttempt $attempt): CertificateEligibility`
           - Check if attempt passed
           - Create CertificateEligibility record linked to attempt
           - Return eligibility record
         - `revokeEligibility(CertificateEligibility $eligibility): bool`
           - Set eligible = false, save
         - `getEligibleMembers(Exam $exam): Collection`
           - Query CertificateEligibility where exam_id = exam, eligible = true
           - With user relationship loaded
         - `getUserEligibilities(User $user): Collection`
           - Query CertificateEligibility where user_id = user, eligible = true
           - With exam relationship loaded

    4. **Update: ExamAttempt model**
       - Add relationship: `certificateEligibility(): HasOne` to CertificateEligibility
       - Update ExamGradingService: after setting passed = true, call CertificateService::createEligibility($attempt)

    5. **Update: User model**
       - Add relationship: `certificateEligibilities(): HasMany` through or direct
       - Method: `getEligibleExamsAttribute()` returns exams user is eligible for

    Run `vendor/bin/pint --dirty` after creating files.
