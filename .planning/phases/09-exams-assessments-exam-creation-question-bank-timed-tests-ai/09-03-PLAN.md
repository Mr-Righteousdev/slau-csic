  <action>
    Create database schema and models for exam attempts and answers:

    1. **Migration: Create exam_attempts table**
       - Use `php artisan make:migration create_exam_attempts_table`
       - Fields: `id`, `exam_id` (foreignId, cascade delete), `user_id` (foreignId, cascade delete), `started_at` (timestamp), `submitted_at` (timestamp, nullable), `time_remaining_seconds` (integer, nullable), `total_score` (integer, default 0), `passed` (boolean, default false), `created_at`, `updated_at`
       - Unique constraint on (`exam_id`, `user_id`) to enforce one attempt per user
       - Index on `exam_id`, `user_id`, `passed`

    2. **Migration: Create exam_answers table**
       - Use `php artisan make:migration create_exam_answers_table`
       - Fields: `id`, `exam_attempt_id` (foreignId, cascade delete), `exam_question_id` (foreignId, cascade delete), `answer_text` (text, nullable - for short answer/code), `selected_option_id` (foreignId, nullable, to question_bank_options - for MCQ/TF), `is_correct` (boolean, nullable), `marks_awarded` (integer, default 0), `created_at`, `updated_at`
       - Index on `exam_attempt_id`, `exam_question_id`

    3. **Model: ExamAttempt.php**
       - Use `php artisan make:model ExamAttempt`
       - Fillable: `exam_id`, `user_id`, `started_at`, `submitted_at`, `time_remaining_seconds`, `total_score`, `passed`
       - Casts method: `started_at` (datetime), `submitted_at` (datetime, nullable), `passed` (boolean), `total_score` (integer), `time_remaining_seconds` (integer, nullable)
       - Relationships:
         - `exam(): BelongsTo` to Exam
         - `user(): BelongsTo` to User
         - `answers(): HasMany` to ExamAnswer
       - Scopes: `scopeCompleted($query)` where not null submitted_at
       - Accessor: `getIsCompletedAttribute()` returns !is_null($this->submitted_at)

    4. **Model: ExamAnswer.php**
       - Use `php artisan make:model ExamAnswer`
       - Fillable: `exam_attempt_id`, `exam_question_id`, `answer_text`, `selected_option_id`, `is_correct`, `marks_awarded`
       - Casts method: `is_correct` (boolean, nullable), `marks_awarded` (integer)
       - Relationships:
         - `attempt(): BelongsTo` to ExamAttempt
         - `examQuestion(): BelongsTo` to ExamQuestion
         - `selectedOption(): BelongsTo` to QuestionBankOption (nullable)

    5. **Service: ExamAttemptService.php**
       - Create in `app/Services/`
       - Methods:
         - `startAttempt(Exam $exam, User $user): ExamAttempt` - creates attempt with started_at = now(), checks no existing attempt
         - `submitAttempt(ExamAttempt $attempt): array` - sets submitted_at, calculates score, sets passed, returns score data
         - `saveAnswer(ExamAttempt $attempt, ExamQuestion $examQuestion, array $data): ExamAnswer` - saves answer (text or selected option)
         - `calculateScore(ExamAttempt $attempt): array` - calculates total score, marks per question, determines pass/fail
         - `getUserAttempt(Exam $exam, User $user): ?ExamAttempt` - gets user's attempt if exists

    Run `vendor/bin/pint --dirty` after creating files.
