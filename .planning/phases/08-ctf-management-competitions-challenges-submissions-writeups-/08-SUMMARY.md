## Key Implementation Details

### Flag Security
- Flags stored as SHA256 hash using CtfChallenge::setFlagAttribute mutator
- Plain text never persisted to database
- Verification uses hash('sha256', $flag) === stored hash

### Scoreboard
- Computed via CtfScoreboardService::getScoreboard
- Cached for 1 minute (faster than general leaderboard's 5 min)
- Only counts correct submissions (is_correct = true)

### Point Awards
- Points awarded via GamificationService::awardPoints on correct submission
- Reference type tracks CTF solves
- Integrates with gamification badges
