**1. GamificationService.php** (`App\Services\GamificationService`):
```php
class GamificationService
{
    public function __construct(
        protected PointTransaction $pointTransaction,
    ) {}

    // Award points to a user, log transaction, check badges, check rank upgrade
    public function awardPoints(
        User $user,
        int $points,
        string $reason,
        ?string $referenceType = null,
        ?int $referenceId = null
    ): PointTransaction { ... }
    
    // Deduct points (negative points)
    public function deductPoints(User $user, int $points, string $reason): PointTransaction { ... }
    
    // Check all badges for a user, unlock any newly earned
    public function checkBadges(User $user): Collection { ... }
    
    // Check if user should upgrade rank
    public function checkRankUpgrade(User $user): ?string { ... }
    
    // Award a badge manually (for 'custom' criteria type)
    public function awardBadge(User $user, Badge $badge): UserBadge { ... }
    
    // Revoke a badge from a user
    public function revokeBadge(User $user, Badge $badge): bool { ... }
}
```

Implementation details:
- `awardPoints()`: Create PointTransaction record. Call `checkBadges()`. Call `checkRankUpgrade()`. Return transaction.
- Use `DB::transaction()` wrapper for atomicity (award points + check badges in same transaction).
- `checkBadges()`: Load all badges where criteria not yet earned by user. For each, call `Badge::checkCriteria($user)`. If true AND user doesn't already have badge (via `user_badges`), create UserBadge record AND if badge has points_bonus > 0, recursively call `awardPoints()` with reason = "Badge bonus: {badge->name}". Return Collection of newly earned badges.
- `checkRankUpgrade()`: Get user's current getTotalPoints, derive target rank. If different from $user->rank, update $user->rank and set $user->rank_changed_at = now(). Return the new rank string or null if no change.
- `awardBadge()`: Create UserBadge record, return it.
- `revokeBadge()`: Delete UserBadge record, return bool.


Implementation details:

  ```sql
  SELECT user_id, SUM(points) as total_points
  FROM point_transactions
  GROUP BY user_id
  ORDER BY total_points DESC
  LIMIT :limit
  ```
  Load User records for the IDs, map to array with rank position, name, avatar, total_points. Cache result.
- `getUserRank()`: Check cache key `gamification.leaderboard.rank.{user_id}` with 5-min TTL. If miss, compute user's rank position from all point_transactions. Cache result.
- `getTopUsersSince()`: Same as getTopUsers but with `WHERE created_at >= :since` clause.
- `invalidateCache()`: Clear `gamification.leaderboard.*` cache keys.
