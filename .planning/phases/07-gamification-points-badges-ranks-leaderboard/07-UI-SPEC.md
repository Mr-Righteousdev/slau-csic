## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Badge icon gaps, inline padding |
| sm | 8px | Compact element spacing, table cell padding |
| md | 16px | Default element spacing |
| lg | 24px | Section padding, card padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Leaderboard row height: min 56px for touch-friendly target
- Badge display grid: `gap-4` (16px)

## UI Components Required

### Leaderboard Page
- Rank column with medal icons for top 3 (🥇🥈🥉)
- User avatar, name, points total columns
- Animated rank change indicators (+/- positions)
- Pagination or infinite scroll
- Time filter toggle (All Time default)

### Badge Display
- Grid layout: 3-4 columns responsive
- Badge icon (SVG), name, description
- Earned indicator: "Earned" / locked state
- Badge rarity: Common (gray), Rare (silver), Epic (gold), Legendary (purple)

### Admin Gamification Panel
- Tab navigation: Points / Badges / Ranks
- Points tab: award form (user select, points, reason)
- Badges tab: CRUD table for badge definitions
- Ranks tab: threshold configuration

### User Profile Integration
- Points total display with icon
- Current rank badge
- Earned badges carousel
