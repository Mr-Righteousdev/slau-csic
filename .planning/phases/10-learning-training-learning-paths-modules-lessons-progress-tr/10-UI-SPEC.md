## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon padding, inline spacing |
| sm | 8px | Compact element spacing, table cell padding |
| md | 16px | Default element spacing, form inputs |
| lg | 24px | Section padding, card padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Module list item height: min 64px for touch-friendly target
- Resource card padding: 16px
- Lesson content area: max-width 768px for readability

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 | 1.5 |
| Label | 12px | 500 | 1.4 |
| Heading | 18px | 600 | 1.3 |
| Display (progress) | 48px | 700 | 1.1 |
| Section title | 24px | 600 | 1.2 |

Font sizes used:
- `text-xs` (12px) — labels, small labels
- `text-sm` (14px) — body text, table content
- `text-base` (16px) — form inputs, cards
- `text-lg` (18px) — section headings
- `text-2xl` (24px) — page titles
- `text-4xl` (48px) — progress percentage

## UI Components Required

### Training Catalog Page (Member)
- Filter bar: category, difficulty, status (enrolled/not enrolled)
- Search input with icon
- Training card grid: thumbnail, title, category badge, difficulty badge, duration, enroll button
- Pagination or infinite scroll
- Enrolled indicator on cards

### Training Detail Page
- Hero section: title, description, category/difficulty badges, duration, instructor
- Progress bar (for enrolled users)
- Module list accordion:
  - Module title, duration, completion status
  - Expand to show lessons
  - "Mark as Complete" button per module
- Resource library section: PDF, video, link cards
- Action buttons: Enroll / Continue / View Certificate

### Learning Path View (Enrolled)
- Sidebar: Module navigation with progress indicators
- Main content: Lesson content viewer
- Navigation: Previous / Next buttons
- "Mark Module Complete" button
- Progress percentage display

### Admin Training Management
- Table: title, category, difficulty, enrollments, status
- Actions: Create, Edit, Delete
- Publish/Unpublish toggle
- View enrollments per training

### Progress Tracking Display
- Overall progress bar by percentage
- Module completion checklist
- Time spent tracking
- Certificate eligibility status

### Resource Library
- Grid layout: 3 columns desktop, 2 tablet, 1 mobile
- Resource cards: type icon (PDF/video/link), title, description
- Filter by resource type
- Search capability
