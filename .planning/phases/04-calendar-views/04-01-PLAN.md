# Phase 4 Plan: Calendar Views

## Objective

Enable month, week, and day calendar views with navigation and "Today" quick link.

## Requirements

- **CAL-01:** Calendar shows month view with event indicators
- **CAL-02:** Calendar shows week view with time slots
- **CAL-03:** Calendar shows day view for single day
- **CAL-04:** Calendar has navigation and "Today" quick link

## Tasks

### Task 1: Enable Week and Day Views

<read_first>
- app/Livewire/EventCalendar.php (existing implementation)
- resources/views/livewire/event-calendar.blade.php (FullCalendar config)
</read_first>

<action>
Update FullCalendar configuration in event-calendar.blade.php to enable all three views:

```javascript
initialView: 'dayGridMonth',
headerToolbar: {
  left: 'prev,next today',
  center: 'title',
  right: 'dayGridMonth,timeGridWeek,timeGridDay'
}
```

The existing EventCalendar.php passes events to the view. No PHP changes needed - the FullCalendar JavaScript library handles views natively.

Update the JavaScript config to include:
- `dayGridMonth` (month view - already default)
- `timeGridWeek` (week view with time slots)
- `timeGridDay` (day view)

Add plugins for timeGrid views if not already present.
</action>

<acceptance_criteria>
- event-calendar.blade.php contains 'timeGridWeek' in the calendar config
- event-calendar.blade.php contains 'timeGridDay' in the calendar config
- All three view buttons appear in the header toolbar
</acceptance_criteria>
